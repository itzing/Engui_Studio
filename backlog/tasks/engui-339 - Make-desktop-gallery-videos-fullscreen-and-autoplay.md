## Problem

Desktop gallery video preview leaves unnecessary inset space in fullscreen viewing and requires manual playback after opening or navigating to a video.

## Acceptance Criteria

- [x] Desktop gallery fullscreen viewer renders videos across the full available viewport.
- [x] Gallery videos start playing automatically when opened.
- [x] Gallery videos start playing automatically after navigating to a video.
- [x] Existing image/audio gallery viewer behavior is unchanged.
- [x] Production build passes.

## Notes

Scope is desktop gallery preview/viewer only. Mobile gallery video behavior is not changed in this task.
