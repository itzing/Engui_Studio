## Description

<!-- SECTION:DESCRIPTION:BEGIN -->
Build the desktop Scene Manager UI as a dedicated full-canvas manager similar in product weight to the larger existing managers. The desktop UI must provide a searchable scene library, single/duo/trio filters, active/trash views, create/edit/duplicate/delete/restore flows, explicit character slot binding, pose and vibe pickers, scene instructions, and an assembled prompt editor with build/rebuild/copy actions. The generatedScenePrompt should remain visible and manually editable, while rebuild from linked presets should stay explicit and protect the user from overwriting manual edits without confirmation. This ticket is desktop-only and should not include mobile Scene Manager UX.
