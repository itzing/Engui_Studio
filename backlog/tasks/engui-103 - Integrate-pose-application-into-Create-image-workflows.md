## Description

<!-- SECTION:DESCRIPTION:BEGIN -->
Integrate Pose Manager into Create for image workflows. On desktop, users must be able to open the full Pose Manager from Create, choose a pose preset, and apply its posePrompt into the active image prompt without destructive overwrite by default. On mobile, this ticket should provide only a lightweight pose picker/apply flow for existing presets, not the full manager UI or editor.
